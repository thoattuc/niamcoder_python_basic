* Dict:
- Dictionary is a collection of objects/items
- Stored as a Key/Value pair
- Disorderly
- Initialization:
    data = {}
    data = {"key": "value",...}

- Key is only one
- Key is immutable
- keyname is distinguished between upper and lower case letters
- Cannot use list is key, list is mutable

- Value is unlimited

- mutable: list; dictionary; set
    co the thay doi gia tri sau khi da duoc khoi tao
- immutable: number; boolean; string; tuple
    mot khi da khoi tao thi khong the thay doi gia tri cua chung

- element access:
    dict_name[key_name]
- update:
    dict_name[key_name] = value
- delete:
    removed_item = dict.pop(key_name)
    dict.pop(key_name)
    del data["key_name"]
    data.clear()
- length:
    len(dict)

* Tuple:
- nhieu dac diem giong voi list: co thu tu, chi so, multi data-type
- la bat bien nen KHONG the them/xoa/thaydoi

- tuple:
    tuple_name = (value_1, value_2, ...)

- tuple packing:
    tuple_name = value_1, value_2, ...n
    a, b, c, ..n = tuple_name

- tinh bat bien chi ap dung tuple, ko ap dung voi cac phan tu cua no
