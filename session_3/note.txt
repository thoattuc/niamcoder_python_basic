    - Tai su dung
    - Clean code
    - def (define)

* Dinh nghia ham:
    def function_name([parameters]):
        statements
        [return value] #thoat khoi ham khi gap return

* Goi ham:
    if __name__ == '__main__':
        value = function_name([arguments])
        print(value)

    - neu ham co tham so can tao bien luu gia tri tra ve

    - default argument: doi so mac dinh

* Cac ham xu ly voi list:
    - insert(): them mot phan tu vao 1 vi tri cu the trong list
    - append(): them mot phan tu vao cuoi list
    - extend(): them tat ca cac phan tu trong 1 iterable vao cuoi list
    - pop(): xoa 1 phan tu o vi tri cu the khoi list
    - remove(): xoa phan tu dau tien voi gia tri cu the khoi list
    - clear(): xoa tat ca cac phan tu khoi list
    - index(): tra ve vi tri cua phan tu dau tien voi gia tri cu the trong list
    - reverse(): dao nguoc thu tu cac phan tu trong list
    - sort()/sorted(): sap xep cac phan tu trong list theo thu tu tang dan
    - count(): dem so lan 1 gia tri xuat hien trong list
